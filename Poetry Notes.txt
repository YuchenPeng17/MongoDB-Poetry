1. Start a poetry project
poetry init

2(Optional, Did Already). Make the virtual environment in/with project
poetry config virtualenvs.in-project true

3. Install
poetry install

4. Run the program inside the virtual environment
4.1 Open a shell within the virtual environment
poetry shell
poetry run python <FILE_NAME.py>
4.2 Exit the shell
exit

5. Add packages to virtual environment
poetry add <package_name>

6. Remove packages from virtual environment
poetry remove <package_name>
